# 소감

SNS 데이터를 활용해 선거 결과를 예측한다는 접근 방식이 흥미롭게 느껴졌다. 기존에는 여론조사나 출구조사처럼 사람을 직접 대상으로 데이터를 수집한다고 생각했는데 사람들이 자발적으로 남긴 데이터를 분석해 실제 문제를 예측할 수 있다는 점이 인상적이었다. 데이터를 단순히 저장하는 것이 아니라 감성 분석과 머신러닝을 통해 의미 있는 정보로 활용할 수 있다는 점도 흥미로웠다.

다만 트위터 이용자가 전체 유권자를 대표한다고 보기 어렵고, 봇 계정이나 여론 조작 등의 영향도 있을 수 있어 SNS 데이터만으로 결과를 예측하는 데에는 한계가 있을 것이라는 생각이 들었다. 또한 Summary에서 언급된 94.2%의 정확도가 감성 분석 모델의 성능인지, 실제 선거 결과 예측의 정확도인지 다소 모호하게 느껴져 본문에서는 어떤 평가 지표를 사용했는지 확인해 보고 싶었다.

이번 Summary를 읽으며 데이터를 많이 모으는 것보다 데이터의 특성과 한계를 함께 고려하는 과정이 중요하다는 점을 다시 한번 느낄 수 있었다.
