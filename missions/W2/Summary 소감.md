# 소감

SNS 데이터를 활용해 선거 결과를 예측한다는 접근 방식이 흥미롭게 느껴졌다. 기존에는 여론조사나 출구조사처럼 사람을 직접 대상으로 데이터를 수집한다고 생각했는데 사람들이 자발적으로 남긴 데이터를 분석하여 실제 선거 결과를 예측할 수 있다는 점이 인상적이었다. 감성 분석과 머신러닝을 활용해 의미 있는 정보를 추출하고 이를 실제 문제 해결에 활용할 수 있다는 점에서 데이터 분석의 활용 범위가 생각보다 넓다는 것을 느낄 수 있었다.

다만 한편으로는 트위터 이용자가 전체 유권자를 대표한다고 보기 어려울 수도 있다는 생각도 들었다. 특정 연령이나 계층에 사용자가 편중될 수 있고, 정치와 같이 민감한 주제에서는 여론 조작이나 봇 계정 등의 영향도 존재할 수 있기 때문이다. 따라서 SNS 데이터만으로 결과를 예측하기보다는 다른 데이터와 함께 활용한다면 더욱 신뢰도 높은 결과를 얻을 수 있을 것이라는 생각이 들었다.

이번 Summary를 읽으며 데이터를 많이 모으는 것보다 어떤 데이터를 활용하고, 그 데이터의 특성과 한계를 함께 고려하는 과정이 중요하다는 점을 다시 한번 느낄 수 있었다.
