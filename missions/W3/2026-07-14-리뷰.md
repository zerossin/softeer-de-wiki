# 2026-07-14 리뷰

오전에는 지난 시간에 만들어둔 싱글 노드와 멀티 노드 클러스터를 요구사항과 다시 대조해보는 시간을 가졌다. 문서를 문자 그대로 읽으면 host network 모드를 써야 할 것처럼 보이지만 실제로는 Docker Desktop 환경에서 지원되지 않아 오히려 바로 다음 요구사항과 충돌하는 지점을 발견했고 지금까지 선택한 방식이 실제 의도에 더 맞는다는 것을 확인했다.

오후에는 나머지 미션들을 이어서 진행했다. 설정 파일을 이해하는 미션에서는 예상 출력 예시가 실제 Hadoop 버전과 다르게 동작하는 부분을 두 군데 발견해서 직접 확인한 값으로 검증했다. 네임노드 디렉터리를 옮기는 과정에서 클러스터 아이디가 어긋나는 실제 오류도 겪었고 원인을 찾아 고쳤다. 이후 미션들에서는 예제로 제공되는 jar 대신 파이썬으로 Mapper와 Reducer를 직접 짜는 방식으로 넘어가서 워드카운트 감성 분류 평균 평점 상품 리뷰 집계까지 서로 다른 네 개의 데이터셋으로 실습했다.

과제를 구현하는 것 자체보다 왜 이렇게 동작하는지를 계속 확인하려고 했다. 매퍼와 워커가 어떻게 다른지 리듀서로 데이터가 어떻게 전달되는지 컴바이너를 왜 추가했는지를 그때그때 짚었다. 그 과정에서 요구사항에 실제로 있는 것과 없는 것을 구분하는 연습도 할 수 있었다.
