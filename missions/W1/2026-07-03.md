# 2026-07-03 회고

## 리뷰

오늘은 별도의 강의 없이 자율적으로 ETL 과제를 수행하는 시간을 가졌다.

과제에서는 BeautifulSoup를 활용해 웹에서 GDP 데이터를 수집하고, Pandas를 이용해 데이터를 가공하는 ETL 파이프라인을 구현했다. 또한 요구사항에 맞게 JSON 파일로 저장하고, SQLite 데이터베이스에 적재한 뒤 SQL을 이용해 필요한 데이터를 조회하는 기능까지 구현했다. ETL 과정별 로그를 남기며 데이터 엔지니어링의 전체 흐름을 직접 경험할 수 있었다.

오후 3시부터 5시까지는 교환 팀 간 회고를 진행했다. 다른 팀의 과제 진행 방식과 문제 해결 과정을 공유받으며, 같은 요구사항이라도 다양한 접근 방식이 존재한다는 것을 알 수 있었다.
